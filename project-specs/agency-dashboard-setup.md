# Agency Dashboard Portal Setup

## Project Overview
Build a modern agency dashboard portal for the AI agents repository. The page should feel like a control room for selecting agents, tracking active projects, and viewing live task status.

## Requirements
- A live task overview section showing tasks in progress
- A projects section with ongoing project summaries and progress
- An agent roster with individual agent cards
- Department view for agents grouped by department
- Dark/light mode toggle with sun/moon icon
- Modern, minimal, responsive layout
- Use Tailwind CSS + clean HTML and vanilla JavaScript
- One single HTML file is acceptable

## Deliverables
- `project-specs/agency-dashboard-setup.md`
- `project-tasks/agency-dashboard-tasklist.md`
- `agency-dashboard.html`
- Responsive dashboard UI with dark/light mode
- Clean interface that can serve as a portal for choosing agents and monitoring work
